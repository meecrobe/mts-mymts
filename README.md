# starter
Simple sass project starter using bower and grunt.

Use ```$ grunt``` command for developing and ```$ grunt release``` to release a project.

```
.
├── dev/
│	├── js/
│	│	├── vendor/
│	│	│	├── modernizr.min.js
│	│	│	└── jquery.min.js
│	│	└── main.js
│	├── sass/
│	│	└── main.sass
│	├── img/
│	├── fonts/
│	└── index.html
├── release/
│	├── js/
│	│	├── vendor/
│	│	│	├── modernizr.min.js
│	│	│	└── jquery.min.js
│	│	└── main.min.js
│	├── css/
│	│	└── main.min.css
│	├── img
│	├── fonts
│	└── index.html
├── bower.json
├── Gruntfile.js
└── package.json
```
